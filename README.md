# MoviesProject---Vuejs

Projet LP Dawin :

Catalogue de films réalisé en Node.js et Vue.js 

## Spécifications

https://scleriot.github.io/web-dawin/project/alternants.html

## Configuration et démarrage

# 1) Exécuter 
    $ npm run prod
    
## OU

# 2) Executer

    $ npm install
    
 puis 

- dans un premier terminal
  ```
  $ npm run server
  ```

- dans un deuxème terminal
  ```
  $ npm run start
  ```
